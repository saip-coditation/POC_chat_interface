How to get your API Token

How to connect your Stripe account:

  1. Create a Stripe account if you don't have one.
    Sign up for Stripe
  2. Log in and go to the Stripe Dashboard → API Keys section.
  3. Locate your "Secret key" (this acts as your authentication token).
  4. Click "Reveal test key" or "Reveal live key".
  5. Copy the key (starts with sk_test_ or sk_live_).
  6. Paste the key into the API Secret Key field below.

⚠️ Use test keys for testing. Never share your live keys publicly.

Quick Connect (Recommended):

  1. Step 1: Go to Zoho API Console
  2. Step 2: Click "Add Client" → select "Self Client"
  3. Step 3: In "Generate Code" tab, enter scope: ZohoCRM.modules.ALL
  4. Step 4: Set duration (10 min) and click "Create"
  5. Step 5: Copy the Authorization Code and paste it below

Found in Zoho API Console under "Client Secret" tab

✅ We'll automatically exchange this for a refresh token!

Manual Method:

  1. Step 1: Go to Zoho API Console
  2. Step 2: Click "Add Client" → select "Self Client"
  3. Step 3: In "Generate Code" tab, enter scope: ZohoCRM.modules.ALL
  4. Step 4: Set time duration (e.g., 10 minutes) and click "Create"
  5. Step 5: Copy the generated code
  6. Step 6: Go to "Generate Token" tab, paste the code, click "Create"
  7. Step 7: Copy the Refresh Token and paste it below

⚠️ Backend needs ZOHO_CLIENT_ID and ZOHO_CLIENT_SECRET in .env

How to connect your GitHub account:

  1. Go to GitHub → Settings → Developer settings → Personal access tokens.
  2. Click "Generate new token (classic)".
  3. Give it a descriptive name like "DataBridge AI".
  4. Select the repo scope (for full repository access).
  5. Click "Generate token" and copy it immediately.

⚠️ Store your token securely. You won't be able to see it again!
